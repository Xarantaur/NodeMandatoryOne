Git i terminal

git add -A, tilføjer alle ændringer der er foretaget i projectet til næste commit

git add ., tilføjer alle mapper til det næste commit.

git commit -M"", commiter de ændringer der er lavet i lokalt repo, -m"" tilføjer en meddelse til commit, det er vigtigt med gode commit meddelser

git push, skubber commitet ud til remote repo på github.

Type Coercion i javascript

Type coercion er en design filosofi i javascript,man har valgt at det skal være muligt i da man designede sproget men hvad er det? det muligegøre at sammenligne forskellige datatyper fordi et sprog som JS tillader det. det laver automatisk en datatype om til den type den skal sammenlignes med så de KAN sammenlignes det er noget vi som udgangspunkt gerne vil undgå, så derfor bruger vi altid strict equality checks " === og !== " eksempel:
