Scoping

  • Globale Scope

    • er alt udenfor functioner og og blocks. Hvis der ligger variabler her så kan alle tilgå dem.

  • Function Scope

    • er området imellem en funktions tuborg klammer {variabler herinde kan kun tilgås i funktionen. }

  • Block Scope

    • en block er ting imellem tuborgklammer {} eksempelvis et If statement eller en forloop. Som ikke har en funktion foran sig.

Express

hvad er Express for noget? et minimalistisk web framework for node.js er beskrivelsen på Express egen hjemmeside Express stiller et library til rådighed med nogle gode værktøjer til at bygge "robuste" API'er hurtigt og nemt. som de selv skriver. www.expressjs.com

før man kan bruge express i sit projekt. skal det installeres og importeres

  • Node_modules

    • når man vil bruge andre frameworks i node.js eksempelvis express skal man installere dette ved hjælp af npm (node package Manager) man kan gøre dette ved at skrive "npm install" i sin terminal. forudsat at man har en package.json fil i sit projekt der specificere hvilke dependencies man gerne vil have ud af node package manageren. når man køre npm så henter den en folder kaldet "node_modules" der indeholder alle de filer og dependencies so det framework man henter skal bruge for at kunne køre

  • Package.json

    • man kan selv oprette en fil der hedder "package.json" med de dependencies man skal bruge. alternativt kan man skrive "npm install express" i sin terminal så opretter package manager selv en package.json fil med en dependency til det specificerede framework. package.json kan sammenlignes med pom.xml i maven projekter

    Server instans med express

    med commonJS "importeres" express in i projekt filen med "require()" express instancieres i const app og den bruges i bunden af siden til at lytte på port 8080 (localhost:8080)
